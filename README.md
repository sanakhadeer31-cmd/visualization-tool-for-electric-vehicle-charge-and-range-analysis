EV Market & Charging Infrastructure Analysis

This project analyzes Electric Vehicle (EV) performance and charging infrastructure using Tableau Public. The analysis focuses on comparing EV brands based on range, price, efficiency, top speed, and powertrain type, along with evaluating charging station distribution across regions in India.
Interactive dashboards were created to highlight market trends, regional infrastructure gaps, and performance differences between EV models.


Tools Used: Tableau Public, CSV Datasets
Key Skills: Data Visualization, Aggregation, Calculated Fields, Dashboard Design
